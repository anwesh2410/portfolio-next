# Ale Anwesh — AI Engineer Portfolio

Modern, responsive, SEO-friendly portfolio built with **Next.js 14**, **TypeScript**, **Tailwind CSS**, and **React Three Fiber** for the animated topographic background.

🔗 Live: [anwesh.me](https://anwesh.me)

## Stack

- **Framework:** Next.js 14 (App Router) · React 18 · TypeScript
- **UI / Styling:** Tailwind CSS · shadcn/ui · Framer Motion · AOS (Animate on Scroll)
- **3D / WebGL:** React Three Fiber · Drei · Postprocessing
- **Theme:** next-themes (dark default)
- **Analytics:** Vercel Analytics
- **Icons:** Lucide · React Icons · Simple Icons CDN

## Local dev

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

Build:

```bash
npm run build
npm run start
```

## Deploy

Push to `main` → auto-deploys on Vercel. Custom domain `anwesh.me` is configured in Vercel's Domain settings.
